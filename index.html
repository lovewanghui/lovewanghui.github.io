checkout, revert, restore, reset

2021-05-26
计算机基础

如果时光可以倒流,你会怎么样? 我相信每个人的人生选择都将会不同,至少你不会再做出现在来看是错误的选择.
在代码的世界里,版本管理工具给你反悔的机会.

Git中存在的几个区域:

不难git这样设计的目的,一个类似于平行宇宙的概念,他不是简单的备份,而且git追踪的是变化,而不是文件本身. git 中被tracking的文件, 就是在不断的从一个区移到另外一个区,同时文件的状态,被标识为不同的状态.

比如:git add指令会把working里面的所有文件,“移动“到staged区,同时working里面所有untracked的文件转换为staged.

我们可以逻辑上,把git里面的文件抽象为以下几种状态,其实这样划分严格意义上并不是很准确,只是方便理解.

Git中文件抽象为以下几种状态及其对应区域:
  • untracked :文件不受git “追踪”的状态,对应working区,你修改后的文件,如果没有add,commit,最开始就是在working区.不难理解working就是当前工作区.
  • modified :git ”追踪“到已经修改,对应working区.
  • staged :git已经暂存,对应staged区.
  • commited :git已经提交,对应repository区.

git命令每次执行完毕, 都会反馈给你一些信息,这些信息包含了,当前分支,文件状态,已经当前命令的回撤和进一步操作的info.你读懂了会发现其实命令行界面也可以用户体验很好.

所以我们知道了,git 命令其实就是让git中tracking的文件, 从一个区“到”另外一个区,同时伴随着文件的状态从一个状态切换到另外一个状态.
其实我们人生活中地点的变化也往往伴随着身份的变化,当我们从家里到公司,身份从儿子/丈夫到工程师.我们从淘宝买的东西,快递在仓库是未发货状态,接着伴随地点的变化,状态变为分拣,运输中,最后到你手里,你确认收货,那就是签收状态了.

如果搞清楚了git中文件的几种状态,就可以看看git中几条让时光逆转的几条指令.

几条时光逆转的指令
  1. git checkout
  2. git restore
  3. git revert
  4. git reset

文件在git中分为不同的状态,以上几条指令可以在不同的状态下切换,达到回退的目的.
具体详细的使用请执行:git help

git checkout 让staged区文件back到working区.覆盖当前working区,从而达到回退目的.git restore 也可以达到目的, 每次add命令执行完毕,都会给出反馈: (use “git restore …” to discard changes in working directory).就是这个意思.

git restore 除了可以让你放弃当前working区的更改,还可以把文件从staged区back到working区.但是文件的改动此时依然存在.

使用restore命令,放弃working区更改&把staged区文件back到working区:

git revert commit-id, 针对repository区的commited 文件, 回退到上次committed,然后把当前回退的改动再次提交. 作为更改它是有记录,更安全.

如果你当前working区, 有改动你想revert,必须要先commit才能revert.

revert 报错

git reset,用来设置分支的头部指向,对应repository区里面有很多提交,使用git reset可以把当前提交重置到某次提交.
reset 有 –soft,–mixed,–hard可选.
当使用hard, 被回退的那些版本的修改会直接舍弃,commit-id所对应的提交直接会覆盖到working区, 此时不管working区里面的文件是什么状态,好像它们没有来过一样。其他的分别会回退到staged区和working区.

总结:写的比较长.感觉还是有不够透彻的地方,凡事都要自己动手.

戒烟后的感受

2021-05-19
生活

戒烟已经差不多三个月了,现在看见别人抽烟不会有想抽的感觉. 但是大脑中依然记忆着曾经抽烟的那种快感. 如果尝试去品味这种快感,尤其是心情不愉悦的时候, 这时候你如果有向烟草寻求帮助的想法,并在大脑中回味那种抽烟之后带来的快感,基本上你很难不来上一根.

抽烟有害健康这是肯定的,但是我不认同抽烟无益,往大了说,任何东西,能存在必然有其一定的价值, 往小了说,抽烟能在你需要的时候暂且让你爽一下,给你短暂的快感.但是这种快感是透支生命的快感作为代价的.

我为什么要戒烟?

主要是我抽烟后呼吸道不是很顺畅,鼻子也不透气, 刚开始我还不能排除是抽烟造成的,后来,我就强忍着一周不抽烟,发现在这一周里,这种感觉逐渐减轻.最终我决定彻底戒掉,引用一个戒烟的前辈的话说:烟草是奴役人的东西,抽烟的人终将不自由,这有悖于我的追求.

戒烟后的感觉?

我只能说感觉很好,整体而言,比以前更快乐了吧.
对自己来说也是一个正反馈,更坚定自己一定要把烟戒掉.
但是抽烟也没什么太大不了的事情,只是我不适合抽烟吧,毕竟人生大几十年,戒这个,戒那个的,对我们普通人来说的确少了点意思.掌握好度, 找到适合自己的就好.

戒烟最艰难的是什么?

  • 饭局上,喝点酒,这时候,某个抽烟的人,给你一支烟,并要给你点上.
  • 心情郁闷,工作/生活遇到阻碍,找不到突破口的时候,就像来一根烟,来舒畅一下.哪怕事实上, 这压根儿,从根本上解决不了任何实质问题.

戒烟是自己想做的, 并且做到了, 接下来就要努力健身了. 争取一年时间把身体恢复到高中时候的那种状态.

岁月静好,
仨瓜俩枣,
愿你一切如初.

计算机基础
在开始之前先谈谈标题.

解释什么是十六进制字符串?
进制是计数方式,利用这方式，可以使用有限的符号来表示所有的数值。
我们知道,在强类型的计算机语言里,数字和字符串,是两种类型,但可不是风马牛不相及.后续我们慢慢道来,是怎么相及的.
也就是说十六进制字符串表示的是一个数,但是呢.这个数是用十六进制这种进制(计数方式)来表示的.

如何把这个“字符串格式的数”convert 一下. 变成一个数字类型的数.
常见的把它变成我们常见的“十进制类型”的数,还有一种是把它转换成“十六进制类型的数”.
本质上这是一种类型转换&保留原来的十六进制.

2.十六进制字符串 转换为 十六进制数字

先不要思考为什么要么做. 但是这么做是有意义的,计算机语言里十六进制的字符串,虽然它表示的是一个数字,但是它的类型还是字符串.我们要把字符串转换成数字类型.

懵了?
别懵,这个不难理解.比如说:int a = 1000,变量a的值是1000,变量a的类型是整形.再看另一个:String a = “1000”,这里a变量的值也是“1000”,a变量的类型是String类型.可能你看起来,区别不大,事实上,对计算机语言来说,区别大了.
到底有什么区别呢?
有空再聊这一块.记着这两种值都是1000,但是类型不一样,对计算机来说,他们区别大了.所以我们要转换.

既然对计算机来说,区别大了,那么各程序设计语言肯定回提供一些方法或者函数来供我们调用来去完成这个convert
